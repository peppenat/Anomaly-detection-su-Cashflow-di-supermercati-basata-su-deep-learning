# Anomaly-detection-su-Cashflow-di-supermercati-basata-su-deep-learning
Pipeline di anomaly detection forecasting-based con LSTM per flussi di cassa sintetici di supermercati. Due casi di studio: level shift nelle vendite e ritardi di settlement POS, con valutazione event-level, analisi di robustezza e confronto con baseline e autoencoder.
